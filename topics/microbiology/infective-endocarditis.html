INFECTIVE ENDOCARDITIS

DEFINITION

Infective endocarditis (IE) refers to microbial invasion of the heart valves or mural endocardium-characteristically results in formation of bulky friable vegetations, composed of mass of platelets, fibrin, microcolonies of organisms, and scanty inflammatory cells.

VEGETATIONS

TABLE: | Feature | Details | | Nature | Bulky, friable | | Composition | Platelets + fibrin + microcolonies of organisms + scant inflammatory cells | | Most common site | Heart valves | | Other sites | Low-pressure side of ventricular septal defect | | | Mural endocardium |

CLASSIFICATION

TABLE: | Feature | Acute IE | Subacute IE | | Evolution | Rapid | Slow | | Valve involved | Normal cardiac valve | Previously damaged (scarred/deformed) valve | | Organism virulence | High | Low | | Example organism | Staphylococcus aureus | Viridans streptococci | | Clinical course | Causes substantial morbidity and mortality even with appropriate antibiotic therapy and/or surgery | Follows gradually progressive course of weeks to months | | Outcome | Severe | Most recover after antibiotic therapy | | Frequency | Less common (10–20%) | More common (50–60%) |

MNEMONICS

  • ACUTE → “RAPID NORMAL DEADLY AUREUS”
  • SUBACUTE → “SLOW DAMAGED RECOVER VIRIDANS”

PATHOGENESIS

FLOWCHART: Predisposing factors -> [Underlying cardiac defect (e.g. mitral regurgitation) | Use of intravenous catheter | Prosthetic valve surgery] -> Abnormal blood flow (turbulence) -> Mechanical stress on endocardium -> Direct endothelial damage -> Loss of endothelial integrity -> Exposure of subendothelial collagen and extracellular matrix -> Platelet adhesion -> Platelet activation -> Fibrin deposition -> Formation of sterile platelet-fibrin thrombus -> NONBACTERIAL THROMBOTIC ENDOCARDITIS (NBTE) -> Transient bacteremia (e.g. after brushing teeth) -> Microorganisms reach NBTE -> Adhesion to thrombus (via adhesins such as dextran) -> Colonization -> Proliferation of organisms -> Entrapment in fibrin mesh -> Protection from immune system -> Continued platelet + fibrin deposition -> Minimal inflammation -> Progressive enlargement -> Formation of infective vegetations -> Persistent bacteremia -> Fragmentation -> Septic emboli -> [Brain | Kidneys | Spleen | Skin | Peripheral tissues] -> Metastatic infection + infarction -> Peripheral signs (petechiae, Janeway lesions, Osler nodes) -> Immune complex formation -> Deposition in tissues -> Glomerulonephritis + Rheumatoid factor positivity

PREDISPOSING FACTORS

TABLE: | Category | Factors | | Cardiac | Underlying cardiac defect (e.g. mitral regurgitation) | | Iatrogenic | Intravenous catheter | | Surgical | Prosthetic valve |

LABORATORY FINDINGS (M FOR LAB)

  • Anemia
  • Leucocytosis
  • Microscopic hematuria
  • Elevated ESR
  • Elevated CRP
  • Rheumatoid factor

IMPORTANT NOTE

  • Majority of Staphylococcus aureus causing IE are MRSA

DIAGNOSIS (MODIFIED DUKE CRITERIA)

OVERVIEW

  • Modified Duke criteria is a highly sensitive and specific diagnostic schema.
  • Based on clinical, laboratory, and echocardiographic findings.

BLOOD CULTURES

  • Isolation of causative microorganism is critical for diagnosis.
  • Helps in antimicrobial susceptibility and treatment planning.
  • Blood cultures should be collected before antibiotic therapy.

COLLECTION

TABLE: | Method | Details | | Method 1 | Two blood culture sets → 12 hr interval | | Method 2 | ≥3 sets → within 24 hr | | Example | 30 min gap between consecutive sets |

BLOOD CULTURE SET

  • Blood collected from single venipuncture site.
  • Divided into two bottles.
  • Considered positive if any bottle flags positive.

INTERPRETATION

  • Major criterion → Typical IE organism from two separate cultures OR persistent isolation
  • Minor criterion → Positive culture not fulfilling major criterion

NON–BLOOD-CULTURE TESTS

TABLE: | Test Type | Details | | Serology | Brucella, Bartonella, Legionella, Chlamydia psittaci, Coxiella burnetii | | Culture | Isolation from vegetations | | Microscopy | PAS stain → Tropheryma whipplei | | Immunologic | Direct fluorescence antibody | | Molecular | PCR → DNA / 16S RNA → identification |

MODIFIED DUKE CRITERIA

MAJOR CRITERIA

1. POSITIVE BLOOD CULTURE

  • Typical IE organism from two separate blood culture sets
  • Persistently positive blood culture
  • Coxiella burnetii (single positive culture or Phase I IgG > 1:800)

2. ENDOCARDIAL INVOLVEMENT

  • Echocardiography: Oscillating intracardiac mass, abscess, prosthetic valve dehiscence
  • New valvular regurgitation

MINOR CRITERIA

TABLE: | Criterion | Details | | Predisposition | Heart condition or IV drug use | | Fever | ≥ 38°C | | Vascular | Emboli, infarcts, aneurysm, hemorrhage, Janeway lesions | | Immunologic | GN, Osler nodes, Roth spots, RF | | Microbiologic | Positive culture not meeting major |

DIAGNOSIS

TABLE: | Combination | Interpretation | | 2 Major | Definite IE | | 1 Major + 3 Minor | Definite IE | | 5 Minor | Definite IE |

EXCLUSION

  • Single positive cultures
  • Coagulase-negative staphylococci
  • Diphtheroids
  • Common contaminants
  • Gram-negative bacilli → rarely cause IE

ECHOCARDIOGRAPHY

  • Confirms anatomical involvement
  • Assesses vegetation size
  • Detects intracardiac complications
  • Evaluates cardiac function

Diagnosis Summary

FLOWCHART: Suspected IE -> Blood cultures (before antibiotics) -> Positive -> Apply Duke criteria -> Major + Minor evaluation -> Definite IE diagnosis Negative -> Non-blood tests -> Serology / PCR / Imaging -> Apply Duke criteria -> Diagnosis

  • S. aureus → “6 + Add-ons in Prosthetic”

  • HACEK → “4 weeks, 6 if prosthetic”
