COMPARATIVE FEATURES OF ANTIDEPRESSANTS

FLOWCHART: Different classes of antidepressants differ in -> [Mechanism of action | Adverse effects | Sedation, weight gain and sexual dysfunction | Drug interactions and special clinical uses]

1. TRICYCLIC ANTIDEPRESSANTS (TCAs)

Examples: Amitriptyline, Imipramine, Nortriptyline, Clomipramine

TABLE: | Feature | Details | | Mechanism of action | Inhibit reuptake of serotonin and noradrenaline into the neuron | | Type | Serotonin and noradrenaline reuptake inhibitors | | Additional actions | Block muscarinic, α-adrenergic and H1 receptors | | Important result | Increase serotonin and noradrenaline at receptors in CNS | FLOWCHART: TCA blocks reuptake of norepinephrine and serotonin -> The amount of both neurotransmitters in the synapse increases -> Mood improves after 2–3 weeks TABLE: | Important Features of TCAs | | Anticholinergic side effects present | | Postural hypotension occurs | | Sedation occurs | | Weight gain may occur | | May precipitate convulsions | | Cardiac arrhythmias may occur |
Mnemonic
"TCA = Tired, Constipated and Arrhythmias"

2. SELECTIVE SEROTONIN REUPTAKE INHIBITORS (SSRIs)

SSRIs selectively inhibit the serotonin transporter (SERT).

FLOWCHART: SSRI blocks serotonin transporter (SERT) -> Reuptake of serotonin into the neuron decreases -> The amount of serotonin in the synapse increases -> Serotonergic activity in the CNS increases -> Depression improves TABLE: | SSRI | Important Feature | | Fluoxetine | Longest acting SSRI; half-life 48–72 hours | | Sertraline | Produces active metabolite; fewer drug interactions | | Fluvoxamine | Effective in OCD | | Citalopram | Less drug interactions | | Escitalopram | Less drug interactions | | Paroxetine | More sexual dysfunction | | Dapoxetine | Rapid action; used in premature ejaculation |

General Features of SSRIs

TABLE: | Feature | Present / Absent | | Anticholinergic effects | Absent | | Hypotension | Absent | | Sedation | Absent | | Weight gain | Absent | | Convulsions | Usually do not precipitate | | Cardiac arrhythmias | Absent | | Cognitive impairment | Absent | | Psychomotor impairment | Absent | | Oral activity | Present |
Mnemonic
"SSRIs are SAFE"
  • S = Sedation absent
  • A = Anticholinergic effects absent
  • F = Fewer cardiac effects
  • E = Effective orally

Important Pharmacokinetic Features of SSRIs

TABLE: | Drug | Special Point | | Fluoxetine | Longest acting SSRI | | Fluoxetine and Sertraline | Produce active metabolites | | Sertraline, Citalopram, Escitalopram | Less likely to cause drug interactions |

Adverse Effects of SSRIs

TABLE: | System | Adverse Effects | | Gastrointestinal | Nausea, vomiting, diarrhoea | | CNS | Headache, insomnia | | Sexual | Sexual dysfunction, impotence, loss of libido | | Drug interactions | Inhibit drug-metabolizing enzymes | FLOWCHART: SSRI is administered -> Serotonin increases in the CNS and GIT -> The patient develops nausea, vomiting and diarrhoea
Mnemonic
"SSRIs give the 3 S problems"
  • S = Stomach upset
  • S = Sleep disturbance
  • S = Sexual dysfunction

3. SEROTONIN AND NORADRENALINE REUPTAKE INHIBITORS (SNRIs)

Examples: Venlafaxine, Duloxetine

FLOWCHART: SNRI blocks reuptake of serotonin and norepinephrine -> Both neurotransmitters increase in the synapse -> Mood and anxiety improve TABLE: | Drug | Important Uses | | Venlafaxine | Depression, anxiety, fibromyalgia | | Duloxetine | Depression, anxiety, fibromyalgia | TABLE: | Feature | Present / Absent | | Anticholinergic effects | Absent | | Sedation | Minimal | | Weight gain | Usually absent | | Convulsions | Usually absent | | Oral effectiveness | Present |

Adverse Effects of SNRIs

TABLE: | Adverse Effect | | Nausea | | Sweating | | Sexual dysfunction | | Anxiety | | Hypertension |
Mnemonic
"SNRIs cause NASAH"
  • N = Nausea
  • A = Anxiety
  • S = Sweating
  • A = Sexual dysfunction
  • H = Hypertension

4. ATYPICAL ANTIDEPRESSANTS

BUPROPION

FLOWCHART: Bupropion inhibits reuptake of dopamine and norepinephrine -> Dopamine and norepinephrine increase in the CNS -> Mood improves and craving for smoking decreases TABLE: | Feature | Details | | Mechanism | Inhibits reuptake of dopamine and norepinephrine | | Important use | Smoking cessation | | Anticholinergic effects | Absent | | Hypotension | Absent | | Sedation | Absent | | Sexual dysfunction | Absent |

Adverse Effects of Bupropion

TABLE: | Adverse Effect | | Dry mouth | | Tremor | | Sweating | | May precipitate seizures |
Bupropion may precipitate seizures.
Mnemonic
"BUPROPION = Breaks smoking But Produces seizures"

MIRTAZAPINE

Mirtazapine is a noradrenergic and specific serotonergic antidepressant (NaSSA).

FLOWCHART: Mirtazapine blocks presynaptic α2 autoreceptors and heteroreceptors -> Release of norepinephrine and serotonin increases -> Antidepressant effect develops

It also blocks H1 receptors.

FLOWCHART: H1 receptor blockade -> Sedation and weight gain occur TABLE: | Feature | Details | | Mechanism | Increases norepinephrine and serotonin release | | Other action | Blocks H1 receptors | | Special point | No sexual dysfunction |

Adverse Effects of Mirtazapine

TABLE: | Adverse Effect | | Sedation | | Weight gain |
Mnemonic
"Mirtazapine makes the patient Munch and doze"

TRAZODONE

FLOWCHART: Trazodone blocks serotonin reuptake -> It also antagonizes 5-HT2 receptors -> Depression improves

Trazodone also blocks α-adrenergic receptors.

FLOWCHART: α-Blockade -> Hypotension develops TABLE: | Feature | Details | | Mechanism | Blocks serotonin reuptake and antagonizes 5-HT2 receptors | | Other action | Blocks α-adrenergic receptors | | Additional effect | Has antianxiety action |

Adverse Effects of Trazodone

TABLE: | Adverse Effect | | Sedation | | Hypotension | | Priapism (painful erection) |
Priapism (painful erection) may occur with trazodone.
Mnemonic
"TRAZODONE = Tired and Troubling erection"

MIANSERIN

FLOWCHART: Mianserin blocks presynaptic α2 receptors -> Release of norepinephrine increases -> Antidepressant effect develops TABLE: | Feature | Details | | Mechanism | Increases norepinephrine release | | Anticholinergic effects | Rare | | Cardiac side effects | Rare | | Major effect | Sedation |
Mnemonic
"MIANSERIN = Mainly sedating"
