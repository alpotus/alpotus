CARBAMAZEPINE (IMINOSTILBENE)

Carbamazepine is chemically related to tricyclic antidepressants.

Mechanism of Action

Like phenytoin, carbamazepine delays recovery of sodium channels from the inactivated state.

FLOWCHART: Carbamazepine binds to voltage-gated sodium channels -> Recovery of sodium channels from the inactivated state becomes slow -> Neuronal excitability decreases -> High-frequency firing is suppressed -> Spread of seizure discharge is prevented

Pharmacokinetics

Carbamazepine is absorbed slowly and erratically from the gastrointestinal tract.

TABLE: | Property | Details | | Absorption | Slow and erratic | | Protein binding | Bound to plasma proteins | | Distribution | Distributed throughout the body including CSF | | Metabolism | Metabolized in liver | | Active metabolite | One metabolite retains anticonvulsant activity |

Repeated use of carbamazepine causes enzyme induction.

FLOWCHART: Carbamazepine is administered repeatedly -> Hepatic microsomal enzymes become induced -> Metabolism of carbamazepine increases -> Its own effectiveness gradually decreases

This phenomenon is called autoinduction.

FLOWCHART: Carbamazepine induces hepatic enzymes -> Metabolism of carbamazepine itself increases -> Plasma concentration falls -> Effectiveness decreases

Carbamazepine also reduces the effectiveness of several other drugs.

TABLE: | Drugs Whose Effectiveness is Reduced by Carbamazepine | | Valproate | | Phenytoin | | Lamotrigine | | Topiramate | | Oral contraceptive pills |
Mnemonic
"CARBAMAZEPINE Clears Contraceptives"

Adverse Effects of Carbamazepine

CNS Adverse Effects

TABLE: | Common CNS Side Effects | | Sedation | | Drowsiness | | Vertigo | | Ataxia | | Diplopia | | Blurred vision | | Confusion | FLOWCHART: Carbamazepine depresses CNS activity -> Sedation and drowsiness occur -> Cerebellar function is affected -> Vertigo and ataxia develop

Gastrointestinal Adverse Effects

TABLE: | Gastrointestinal Side Effects | | Nausea | | Vomiting |

Hypersensitivity Reactions

TABLE: | Hypersensitivity Reaction | | Skin rash | | Eosinophilia | | Lymphadenopathy | | Hepatitis |

Rarely, carbamazepine may produce bone marrow depression.

FLOWCHART: Bone marrow function becomes suppressed -> [Neutropenia | Aplastic anaemia | Agranulocytosis] may occur

On chronic therapy, carbamazepine may cause water retention because of ADH release.

FLOWCHART: Carbamazepine increases ADH release -> Water retention occurs -> Body weight may increase and hyponatraemia may develop
Mnemonic
"CARBAMAZEPINE = SAD BRAIN"
  • S = Sedation
  • A = Ataxia
  • D = Diplopia
  • B = Bone marrow depression
  • R = Rash
  • A = ADH release
  • I = Increased water retention
  • N = Nausea

Uses of Carbamazepine

TABLE: | Use | Importance | | Generalized tonic-clonic seizures | Drug of choice | | Partial seizures | Drug of choice | | Trigeminal neuralgia | Drug of choice | | Acute mania and bipolar disorder | Useful |

Trigeminal Neuralgia

FLOWCHART: Trigeminal nerve fires repeatedly at a high frequency -> Carbamazepine suppresses high-frequency discharge -> Pain decreases

Other drugs useful in trigeminal neuralgia include:

TABLE: | Other Drugs Useful in Trigeminal Neuralgia | | Phenytoin | | Gabapentin | | Amitriptyline | | Baclofen |

If drug treatment fails, other options may be used.

TABLE: | Other Treatment Options | | Surgical division of nerve | | Cryosurgery | | Injection of alcohol | | Injection of phenol near the nerve or ganglion |

Carbamazepine is not effective in diabetic neuropathy.

Mnemonic
"Carbamazepine for Convulsions and Cranial pain"

PHENOBARBITONE

Phenobarbitone is a barbiturate. It was widely used earlier, but its use has decreased because safer drugs are available.

Mechanism of Action

FLOWCHART: Phenobarbitone facilitates GABA activity -> Chloride entry into neurons increases -> Neurons become hyperpolarized -> Neuronal excitability decreases -> Seizures are suppressed

Phenobarbitone also has a direct GABA-mimetic action.

Pharmacokinetics

TABLE: | Property | Details | | Absorption | Slow but complete after oral administration | | Protein binding | About 50% | | Enzyme induction | Present | FLOWCHART: Repeated administration of phenobarbitone occurs -> Hepatic enzymes become induced -> The effectiveness of coadministered drugs decreases

Adverse Effects of Phenobarbitone

TABLE: | Adverse Effect | | Sedation | | Nystagmus | | Ataxia | | Confusion | | Megaloblastic anaemia | | Skin rash |

Tolerance to sedation develops gradually.

FLOWCHART: Phenobarbitone is given repeatedly -> Sedation occurs initially -> The body gradually adapts -> Sedation becomes less marked

On chronic therapy in children:

FLOWCHART: Phenobarbitone is used for a long time -> Behavioural disturbances develop -> Memory becomes impaired
Mnemonic
"PHENOBARBITONE = SLEEPY CHILD"
  • S = Sedation
  • L = Learning and memory impairment
  • E = Enzyme induction
  • E = Eye signs such as nystagmus
  • P = Poor coordination (ataxia)

Uses of Phenobarbitone

TABLE: | Use | | Generalized tonic-clonic seizures | | Partial seizures | | Prophylaxis of febrile convulsions | | Status epilepticus when diazepam and phenytoin fail |

Phenobarbitone is the cheapest antiepileptic drug.

ETHOSUXIMIDE

Ethosuximide is mainly useful in absence seizures.

Mechanism of Action

FLOWCHART: Ethosuximide blocks T-type calcium current in thalamic neurons -> Abnormal rhythmic discharge in the thalamus decreases -> Absence seizures are prevented
Mnemonic
"ETHOSUXIMIDE Ends the Empty stare"

Pharmacokinetics

TABLE: | Property | Details | | Absorption | Completely absorbed after oral administration |

Mnemonic
"ETHOSUXIMIDE = Empty stomach and Empty marrow"

BENZODIAZEPINES

The important benzodiazepines used in epilepsy are diazepam, lorazepam and clonazepam.

DIAZEPAM AND LORAZEPAM

Diazepam and lorazepam are mainly used in status epilepticus.

TABLE: | Other Conditions in Which They Are Used | | Tetanus | | Eclamptic convulsions | | Febrile convulsions | | Drug-induced convulsions | FLOWCHART: Intravenous diazepam or lorazepam is given -> GABA activity increases rapidly -> Neuronal excitability falls -> Convulsions stop

Diazepam has a rapid onset but short duration because it is rapidly redistributed.

FLOWCHART: Diazepam is injected -> It rapidly enters the brain -> Seizure control occurs quickly -> The drug redistributes into fat and other tissues -> Its effect becomes short-lived

Therefore repeated doses may be required.

Lorazepam is preferred in status epilepticus.

TABLE: | Reason Why Lorazepam is Preferred | | Longer duration of action | | Less damaging effect on injected vein |
Mnemonic
"LORAZEPAM LASTS LONGER"

FEBRILE SEIZURES

Febrile seizures usually occur in children younger than 5 years of age and are associated with fever.

FLOWCHART: Fever develops in a young child -> Body temperature rises rapidly -> A convulsion occurs

Treatment is done step by step.

FLOWCHART: Febrile seizure treatment -> [Fever is controlled with paracetamol and external cooling | If convulsion occurs] -> [Diazepam 0.5 mg/kg is administered rectally]
